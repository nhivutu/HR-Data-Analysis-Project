# Adidas Retail Sales Performance Analysis

## Project Overview
This project analyzes Adidas retail sales data to identify revenue trends, product performance, and regional sales distribution. The goal of this analysis is to transform raw sales data into meaningful insights that support data-driven business decisions.

Using Excel for data exploration and visualization, the project examines how different products, retailers, and sales channels contribute to overall revenue performance.

---

## Objectives

The objectives of this project are to:

- Analyze Adidas sales performance across regions and retailers  
- Identify top-performing products and product categories  
- Explore revenue distribution across different sales channels  
- Understand regional sales trends  
- Build an interactive dashboard to visualize key performance metrics

---

## Dataset

The dataset contains Adidas retail sales information across multiple regions and retailers.

Key fields include:

- Retailer  
- Region  
- State  
- City  
- Product  
- Price per Unit  
- Units Sold  
- Total Sales  
- Sales Method (Online / In-store)  
- Invoice Date

The dataset provides insights into retail sales performance across geographic regions and distribution channels.

---

## Data Analysis Process

### 1. Data Cleaning
- Checked for missing or inconsistent values  
- Verified data formatting and structure  
- Prepared the dataset for analysis

### 2. Exploratory Data Analysis (EDA)
Explored patterns in the dataset including:

- Sales distribution across regions  
- Product-level sales performance  
- Retailer performance  
- Sales channel trends (online vs in-store)

### 3. Dashboard Development
Built an Excel dashboard to visualize key metrics such as:

- Total revenue  
- Units sold  
- Regional sales performance  
- Product-level insights

---

## Key Insights

Some insights from the analysis include:

- Certain regions generate a significantly higher share of total sales  
- A small number of products drive a large portion of revenue  
- Retailers show different performance levels across regions  
- Online and in-store sales channels contribute differently to total revenue

These insights demonstrate how retail data can support product strategy and sales optimization.
